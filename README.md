# Vue 3 + TypeScript + Vite

## npm install

```
npx create-vite@5.1.0 tutor --template vue-ts
cd tutor
npm install --legacy-peer-deps
npm install vite@6.0.12 @vitejs/plugin-vue@5.0.4 --save-dev --legacy-peer-deps
npm run dev
```
